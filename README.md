# DevelopersHub Internship Projects — Flutter Basics

A collection of my Flutter projects built during a 3‑week internship at **DevelopersHub Corporation**, focusing on UI, navigation, state management, validation, and persistent storage with `SharedPreferences`.

---

## 📁 Projects Overview

### **Week 1 — Basic UI & Navigation**
- Built a **Login Screen** with two `TextFormField`s (email & password) and a **"Forgot Password?"** button.
- Implemented **navigation** to a Home Screen using `Navigator.push()`.
- Added **form validation**:
  - Email: proper email format
  - Password: non-empty validation

### **Week 2 — State Management & Data Persistence**
- Developed a **Counter App**:
  - State managed using `setState`
  - Counter value saved and retrieved using `SharedPreferences`
- Created a **To‑Do List App**:
  - Add new tasks
  - Display tasks in a `ListView`
  - Persist tasks locally with `SharedPreferences`

### **Week 3 — Final Task Manager App**
- Built a **Task Management App** combining all previously learned concepts:
  - Add, delete, and mark tasks as complete
  - Persistent storage using `SharedPreferences`
- Enhanced UI with:
  - Custom **AppBar** with title and action button
  - Icons from Flutter’s built-in `Icons` library
- Tested navigation and data flow using Flutter’s built-in dev tools

---

## 📘 What I Learned

- Flutter UI layout using `Column`, `Row`, `Container`, `ListView`
- Handling forms and validation using `TextFormField` and custom validators
- Navigation between screens using `Navigator`
- State management using `setState`
- Local data persistence using `SharedPreferences`
- Writing structured, maintainable code in Flutter
- Debugging and testing using Flutter’s dev tools
- Building complete, functional apps from scratch
